How to Install Windows OS from Scratch

  1. Download Rufus: Go to rufus.ie.
  2. Get Windows ISO: Download from Microsoft's official site.
  3. Make Bootable: Plug in your pendrive, open Rufus, select the ISO, and click 'Start'.
  4. Boot from USB: Restart PC, tap F12/F11, and select the Pendrive.
