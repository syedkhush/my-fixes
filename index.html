💻 Install Windows OS from Scratch

Follow these steps to clean install Windows 11/10 using a bootable USB.

Official Windows ISO Download Rufus
  1. Prepare USB: Open Rufus, select your ISO, set Partition to GPT, and click Start.
  2. Booting: Insert USB, restart PC, and tap your Boot Key (F12, F11, or Esc).
  3. Install: Choose "Custom: Install Windows only (advanced)".
  4. Partition: Select your Drive 0 and click Next to begin.

🐧 Install Ubuntu Linux (Free OS)

A fast, secure, and open-source alternative to Windows.

Download Ubuntu ISO
  1. Create Media: Use Rufus in "ISO Image Mode" to flash the Ubuntu ISO to your USB.
  2. Live Mode: Boot from USB and select "Try Ubuntu" to test your Wi-Fi and hardware.
  3. Installation: Click "Install Ubuntu" and choose "Normal Installation".
  4. Setup: Select "Erase disk and install Ubuntu" (Warning: This wipes the drive!).

📜 Get Damaged/Lost Certificates (JNTUA)

Process for a Duplicate Consolidated Marks Memo (CMM).

  1. Affidavit: Get a notarized affidavit on ₹50/100 stamp paper explaining the damage.
  2. Application: Visit the JNTUA Examination Portal and fill out the duplicate CMM form.
  3. Fees: Pay the required university fee via Online Challan or DD.
  4. Submission: Submit the affidavit and payment receipt to the Registrar/Director of Evaluation.

🚗 Change a Car Stepney (Spare Tire)

  1. Secure: Park on level ground, pull the handbrake, and use hazard lights.
  2. Loosen: Loosen the wheel nuts slightly before jacking the car up.
  3. Lift: Place the jack under the frame and lift until the tire clears the ground.
  4. Swap: Replace the flat with the stepney and tighten nuts in a star pattern.

⚡ Fix a Damaged Electric Fuse

Warning: Never touch the fuse box with wet hands.

  1. Main Switch: Turn OFF the main power supply completely.
  2. Identify: Pull the fuse carrier and look for the broken or burnt wire.
  3. Replace: Use a new fuse wire or cartridge of the exact same rating (e.g. 16A).
  4. Restore: Push the carrier back in and turn on the main switch.
